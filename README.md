Betting Game System 🎰

A simple betting game system developed in Java using Swing.

The project simulates a slot-style game based on a tic-tac-toe (3x3) board. 
Players place a bet, the symbols spin with animation, and if three identical symbols align
(horizontally, vertically, or diagonally), the player wins a reward.
Features
- Tic-tac-toe style slot game (3x3)
- Animated symbol spinning
- Winning line highlight
- Sound effects for win and lose
- Balance management system
- Simple and intuitive interface

Technologies Used
- Java
- Java Swing (GUI)
- Timer for animations
- Basic game logic and validation

How It Works
1. The player enters a bet amount.
2. The symbols spin with animation.
3. The system checks for winning combinations.
4. If the player wins, the balance increases.
5. If the player loses, the balance decreases.

 Purpose
This project was created for learning purposes, focusing on Java, GUI development,
game logic, and clean code organization.
